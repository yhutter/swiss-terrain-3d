# Swiss Terrain 3D
Implementation of a 3D Terrain Renderer in ThreeJS based on Swisstopo Dataset 

## :package: Libraries
This project uses the following libraries
- [ThreeJS](https://threejs.org/)
- [TweakPane](https://tweakpane.github.io/docs/)

## :rocket: How to Run
```bash
cd src
npm run dev

```
